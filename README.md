
### Vision Systems Project

    This project is to display the basic image processing we can do with a few scripts. 
    The types of processing used here are:

    -RGB Filtering
    -Gamma Correction
    -Histogram Clipping
    -Histogram Gray Filter
    -Edge Detection
    -Corner Detection
